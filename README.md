## JS Game Project
A 2D side-scrolling platform game built using JavaScript and the p5.js library.

### Overview

This project was developed as part of the University of London Programming I course.  
The game features character movement, scrolling background, enemies, collectibles, and win/lose conditions.

### Features

- Character movement (left, right, jump)
- Side-scrolling world
- Obstacles: platforms and terrain
- Collectable items: rings
- Life system (3 tokens)
- Enemies with collision detection
- Sound effects and background music
- Game states: start, game over, win


### Controls

- Move left/right: ← → - side arrows
- Jump: SPACE / ↑ - up arrow
- Start / Restart game: SPACE 

### Technologies Used

- JavaScript  
- p5.js  
- p5.sound  


### How to Run

1. Download or clone the repository  
2. Open "index.html" in your browser
